рЃЕрЃЋрЃћрЃюрЃў рЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА Random Forest рЃарЃћрЃњрЃарЃћрЃАрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃљрЃарЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃарЃў, рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃљ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃБрЃЏрЃЎрЃџрЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌ, рЃўрЃАрЃћ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃА.

(рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ: рЃљрЃЏрЃЪрЃљрЃЏрЃўрЃюрЃЊрЃћрЃџ рЃЋрЃћрЃарЃАрЃўрЃљрЃерЃў рЃћрЃА рЃцрЃљрЃўрЃџрЃў рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ. рЃАрЃљрЃЏрЃљрЃњрЃўрЃћрЃарЃЮрЃЊ, train.csv рЃўрЃДрЃЮрЃцрЃљ рЃџрЃЮрЃЎрЃљрЃџрЃБрЃа рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ, рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљрЃЊ.)
features.csv: рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЌрЃљрЃарЃўрЃдрЃўрЃА, рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА, рЃбрЃћрЃЏрЃърЃћрЃарЃљрЃбрЃБрЃарЃўрЃА, рЃАрЃљрЃгрЃЋрЃљрЃЋрЃўрЃА рЃцрЃљрЃАрЃўрЃА, CPI-рЃўрЃА, рЃБрЃЏрЃБрЃерЃћрЃЋрЃарЃЮрЃЉрЃўрЃА рЃЊрЃЮрЃюрЃўрЃА рЃЊрЃљ MarkDown рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА (MarkDown1-5) рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ. MarkDown рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃ«рЃерЃўрЃарЃљрЃЊ рЃљрЃЦрЃЋрЃА рЃЉрЃћрЃЋрЃарЃў рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
stores.csv: рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃа рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ: рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃў (Type) рЃЊрЃљ рЃќрЃЮрЃЏрЃљ (Size).

рЃћрЃА рЃћрЃбрЃљрЃърЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃА Random Forest рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА.
рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ: рЃЌрЃљрЃЋрЃЊрЃљрЃърЃўрЃарЃЋрЃћрЃџрЃљрЃЊ рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ train.csv, features.csv рЃЊрЃљ stores.csv рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃћрЃарЃЌрЃўрЃљрЃюрЃЊрЃћрЃЉрЃљ train_merged DataFrame-рЃерЃў.

рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ: Date рЃАрЃЋрЃћрЃбрЃўрЃЊрЃљрЃю рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃарЃЮрЃќрЃћ рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃгрЃћрЃџрЃў (Year), рЃЌрЃЋрЃћ (Month), рЃЎрЃЋрЃўрЃарЃљ (Week), рЃЎрЃЋрЃўрЃарЃўрЃА рЃЊрЃдрЃћ (DayOfWeek), рЃгрЃџрЃўрЃА рЃЊрЃдрЃћ (DayOfYear), рЃЌрЃЋрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃў/рЃЉрЃЮрЃџрЃЮ (IsMonthStart, IsMonthEnd) рЃЊрЃљ рЃЊрЃарЃЮрЃўрЃА рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў (TimeIdx).

рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ: Weekly_Sales рЃЋрЃћрЃџрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў (рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃћрЃЉрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃЊрЃћрЃА) рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ 0-рЃўрЃЌ.

рЃњрЃљрЃЏрЃЮрЃбрЃЮрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ (MarkDowns): MarkDown1-5 рЃАрЃЋрЃћрЃбрЃћрЃЉрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў NaN рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃўрЃЋрЃАрЃћрЃЉрЃљ 0-рЃўрЃЌ. рЃћрЃА рЃЏрЃўрЃљрЃюрЃўрЃерЃюрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџ рЃърЃћрЃарЃўрЃЮрЃЊрЃерЃў рЃАрЃљрЃарЃћрЃЎрЃџрЃљрЃЏрЃЮ рЃљрЃЦрЃфрЃўрЃљ рЃљрЃа рЃДрЃЮрЃцрЃўрЃџрЃљ рЃљрЃЦрЃбрЃўрЃБрЃарЃў.

рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ: Type (рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃў A, B, C) рЃАрЃЋрЃћрЃбрЃў рЃњрЃљрЃарЃЊрЃљрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ One-Hot Encoding-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. Store рЃЊрЃљ Dept рЃАрЃЋрЃћрЃбрЃћрЃЉрЃў рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ рЃњрЃљрЃюрЃўрЃ«рЃўрЃџрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў ID-рЃћрЃЉрЃў Random Forest-рЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃў рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ "рЃърЃљрЃарЃљрЃАрЃЎрЃћрЃЋрЃўрЃА рЃњрЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃљ" (Friday Alignment). рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃў рЃќрЃБрЃАрЃбрЃЊрЃћрЃЉрЃљ рЃБрЃљрЃ«рЃџрЃЮрЃћрЃА рЃърЃљрЃарЃљрЃАрЃЎрЃћрЃЋрЃќрЃћ. рЃћрЃА рЃЎрЃарЃўрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃДрЃЮрЃЋрЃћрЃџрЃЎрЃЋрЃўрЃарЃћрЃБрЃџрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃЏрЃљ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃљрЃ«рЃљрЃџрЃў рЃАрЃарЃБрЃџрЃў рЃЎрЃЋрЃўрЃарЃљ, рЃарЃљрЃф рЃ«рЃћрЃџрЃА рЃБрЃерЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃњрЃљрЃЪрЃЮрЃюрЃЋрЃљрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃБрЃцрЃарЃЮ рЃарЃћрЃљрЃџрЃўрЃАрЃбрЃБрЃа рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃљрЃА.

рЃфрЃўрЃЎрЃџрЃБрЃарЃў рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў: рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў: Month_sin рЃЊрЃљ Month_cos. рЃћрЃА рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃћрЃЉрЃў Random Forest-рЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃБрЃЎрЃћрЃЌрЃћрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА, рЃњрЃљрЃўрЃњрЃЮрЃА рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃљ рЃЊрЃљ рЃЌрЃЋрЃћрЃћрЃЉрЃўрЃА рЃфрЃўрЃЎрЃџрЃБрЃарЃў рЃЉрЃБрЃюрЃћрЃЉрЃљ, рЃарЃљрЃф рЃ«рЃерЃўрЃарЃљрЃЊ рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃћрЃА.

Weekly_Sales-рЃерЃў рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЎрЃЋрЃџрЃљрЃЋ рЃюрЃБрЃџрЃЊрЃћрЃЉрЃљ.

MarkDown рЃАрЃЋрЃћрЃбрЃћрЃЉрЃерЃў NaN рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЎрЃЋрЃџрЃљрЃЋ рЃюрЃБрЃџрЃЊрЃћрЃЉрЃљ.

CPI рЃЊрЃљ Unemployment рЃАрЃЋрЃћрЃбрЃћрЃЉрЃерЃў NaN рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃўрЃЋрЃАрЃћрЃЉрЃљ ffill() рЃЊрЃљ bfill()-рЃўрЃЌ рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃў рЃЊрЃљрЃарЃЕрЃћрЃюрЃўрЃџрЃў NaN-рЃћрЃЉрЃў рЃЏрЃћрЃЊрЃўрЃљрЃюрЃўрЃЌ.

рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА Pipeline: ColumnTransformer рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў (Type рЃАрЃЋрЃћрЃбрЃўрЃА One-Hot Encoding) рЃЊрЃљ рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ.

RandomizedSearchCV рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Random Forest-рЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃАрЃљрЃФрЃћрЃЉрЃюрЃљрЃЊ. рЃЊрЃарЃЮрЃўрЃЌрЃў рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ TimeSeriesSplit.

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃљ рЃЏрЃЎрЃљрЃфрЃарЃў рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ» рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃўрЃџрЃўрЃДрЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃў рЃЊрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ.

1. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃгрЃЏрЃћрЃюрЃЊрЃљ

train.csv, stores.csv рЃЊрЃљ features.csv рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃЊрЃљ рЃћрЃарЃЌрЃўрЃљрЃю DataFrame-рЃерЃў.
рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃњрЃљрЃгрЃЏрЃћрЃюрЃЊрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃЊрЃљ рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў IsHoliday рЃЎрЃЮрЃџрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА.

2. рЃЊрЃарЃЮрЃўрЃЌрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃА рЃЊрЃљрЃћрЃЏрЃљрЃбрЃљ рЃљрЃ«рЃљрЃџрЃў, рЃЊрЃарЃЮрЃўрЃЌрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў:

рЃћрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃБрЃюрЃљрЃарЃА, рЃљрЃЏрЃЮрЃўрЃфрЃюрЃЮрЃА рЃЊрЃарЃЮрЃўрЃЌрЃў рЃбрЃћрЃюрЃЊрЃћрЃюрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃћрЃќрЃЮрЃюрЃБрЃарЃЮрЃЉрЃљ.

3. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ

рЃЕрЃљрЃбрЃљрЃарЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃўрЃдрЃарЃЏрЃўрЃАрЃћрЃБрЃџрЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў:
рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃў: рЃњрЃљрЃЏрЃЮрЃЋрЃџрЃўрЃюрЃЊрЃљ рЃЊрЃљ рЃњрЃљрЃљрЃюрЃљрЃџрЃўрЃќрЃЊрЃљ рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃћрЃЉрЃў.
рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ: рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃўрЃџрЃўрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ (рЃ░рЃўрЃАрЃбрЃЮрЃњрЃарЃљрЃЏрЃћрЃЉрЃў) рЃЊрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃў рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.
рЃЊрЃарЃЮрЃўрЃЌрЃў рЃбрЃћрЃюрЃЊрЃћрЃюрЃфрЃўрЃћрЃЉрЃў: рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃБрЃарЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃў рЃЊрЃарЃЮрЃЌрЃљ рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў.

4. рЃљрЃБрЃбрЃџрЃљрЃўрЃћрЃарЃћрЃЉрЃўрЃА (Outlier) рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ

рЃњрЃљрЃЏрЃЮрЃЋрЃџрЃўрЃюрЃЊрЃљ рЃЊрЃљ рЃњрЃљрЃљрЃюрЃљрЃџрЃўрЃќрЃЊрЃљ рЃљрЃБрЃбрЃџрЃљрЃўрЃћрЃарЃћрЃЉрЃў. рЃЏрЃљрЃЌ рЃЊрЃљрЃАрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃљ рЃЉрЃўрЃќрЃюрЃћрЃА-рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃќрЃдрЃЋрЃарЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃўрЃАрЃљ (A, B, C) рЃЊрЃљ рЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (IsHoliday).
A рЃбрЃўрЃърЃўрЃА рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃў: рЃќрЃдрЃЋрЃљрЃарЃў рЃЊрЃљрЃгрЃћрЃАрЃЊрЃљ рЃюрЃЮрЃарЃЏрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
B рЃЊрЃљ C рЃбрЃўрЃърЃўрЃА рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃў: рЃќрЃдрЃЋрЃарЃћрЃЉрЃў рЃЊрЃљрЃгрЃћрЃАрЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃќрЃЮрЃЏрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃЮрЃбрЃћрЃюрЃфрЃўрЃљрЃџрЃўрЃА рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃўрЃЌ.
рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃЊрЃљрЃю рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃљрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃўрЃЌ 16,000-рЃќрЃћ рЃЏрЃћрЃбрЃў рЃљрЃБрЃбрЃџрЃљрЃўрЃћрЃарЃў.

5. рЃГрЃљрЃарЃЉрЃў (Redundant) рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃерЃЮрЃарЃћрЃЉрЃљ

рЃњрЃљрЃљрЃюрЃљрЃџрЃўрЃќрЃЊрЃљ рЃЊрЃљ рЃЏрЃЮрЃерЃЮрЃарЃЊрЃљ рЃГрЃљрЃарЃЉрЃў рЃљрЃю рЃЏрЃљрЃдрЃљрЃџрЃў рЃЎрЃЮрЃарЃћрЃџрЃљрЃфрЃўрЃўрЃА рЃЏрЃЦрЃЮрЃюрЃћ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў (рЃЏрЃљрЃњ., DaysFromStart, WeekOfYear, Quarter, Year, IsQuarterStart/End, Day), рЃарЃљрЃЌрЃљ рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃБрЃџрЃўрЃДрЃЮ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃБрЃџрЃў рЃЊрЃљ рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃфрЃўрЃБрЃџрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў.

6. рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ

Type рЃЎрЃЮрЃџрЃЮрЃюрЃљ (рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА A, B, C рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃА) рЃњрЃљрЃЊрЃљрЃЎрЃЮрЃЊрЃўрЃарЃЊрЃљ рЃарЃўрЃфрЃ«рЃЋрЃўрЃЌ рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў Label Encoding-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃћрЃА рЃћрЃбрЃљрЃърЃў рЃАрЃљрЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃЮрЃЊ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ Random Forest-рЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.
рЃЊрЃарЃЮрЃўрЃЌрЃў рЃњрЃљрЃДрЃЮрЃцрЃљ (Time Series Split)

рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃџрЃљрЃЊ рЃарЃћрЃљрЃџрЃБрЃа рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў, рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЊрЃарЃЮрЃўрЃЌрЃў рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљ:
рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃЮрЃцрЃљ: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃўрЃДрЃЮ рЃЦрЃарЃЮрЃюрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃЌ 80% (рЃбрЃарЃћрЃўрЃюрЃўрЃюрЃњрЃў), 10% (рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ) рЃЊрЃљ 10% (рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ).
рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃњрЃљрЃДрЃЮрЃцрЃўрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃћрЃЉрЃў:
рЃбрЃарЃћрЃўрЃюрЃўрЃюрЃњрЃў: 2010Рѕњ02Рѕњ05 - 2011Рѕњ04Рѕњ01
рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ: 2011Рѕњ04Рѕњ08 - 2011Рѕњ05Рѕњ27
рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ: 2011Рѕњ06Рѕњ03 - 2012Рѕњ10Рѕњ26

Walmart-рЃўрЃА рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљ Random Forest-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ



рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЏрЃўрЃќрЃюрЃљрЃЊ рЃўрЃАрЃљрЃ«рЃљрЃЋрЃА Walmart-рЃўрЃА рЃДрЃЮрЃЋрЃћрЃџрЃЎрЃЋрЃўрЃарЃћрЃБрЃџрЃў рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃќрЃБрЃАрЃб рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃљрЃА. рЃЕрЃЋрЃћрЃю рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ Random Forest рЃЏрЃЮрЃЊрЃћрЃџрЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃўрЃдрЃарЃЏрЃўрЃАрЃћрЃБрЃџ рЃљрЃюрЃљрЃџрЃўрЃќрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљрЃќрЃћ рЃЊрЃљрЃДрЃарЃЊрЃюрЃЮрЃЉрЃўрЃЌ. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃЉрЃўрЃќрЃюрЃћрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў, рЃАрЃљрЃюрЃЊрЃЮ рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.


рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃЊрЃљрЃДрЃЮрЃцрЃљ

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃЊрЃљ рЃАрЃљрЃЏрЃў рЃгрЃДрЃљрЃарЃЮрЃА (train.csv, stores.csv, features.csv) рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљрЃА, рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃбрЃарЃљрЃЦрЃфрЃўрЃљрЃА, рЃљрЃБрЃбрЃџрЃљрЃўрЃћрЃарЃћрЃЉрЃўрЃА рЃЉрЃўрЃќрЃюрЃћрЃА-рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃБрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА (рЃЏрЃљрЃдрЃљрЃќрЃўрЃўрЃА рЃбрЃўрЃърЃў) рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљрЃА.

рЃЏрЃћрЃбрЃарЃўрЃЎрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ: рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ (MAE), рЃцрЃћрЃАрЃЋрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃБрЃџрЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃЊрЃљрЃю (RMSE) рЃЊрЃљ рЃерЃћрЃгрЃЮрЃюрЃўрЃџрЃў рЃАрЃљрЃерЃБрЃљрЃџрЃЮ рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ (WMAE) рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃљрЃбрЃарЃћрЃюрЃўрЃюрЃњрЃЮ рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (in-sample), рЃљрЃАрЃћрЃЋрЃћ рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (out-of-sample).

рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў: рЃњрЃЋрЃљрЃгрЃЋрЃЊрЃўрЃА рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃћрЃЉрЃА рЃарЃЮрЃњрЃЮрЃарЃф рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃўрЃА, рЃљрЃАрЃћрЃЋрЃћ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃерЃў.

рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃЊрЃљрЃцрЃљрЃарЃЋрЃљ: рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃўрЃАрЃљ рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃћрЃарЃўрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃўрЃА рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃћрЃЉрЃА рЃЊрЃљ рЃБрЃгрЃДрЃЋрЃћрЃбрЃЮрЃЉрЃљрЃА.

рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃћрЃЉрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃўрЃќрЃў (рЃбрЃћрЃАрЃбрЃўрЃА рЃюрЃљрЃЎрЃарЃћрЃЉрЃў):

рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ predictions_comparison DataFrame, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃА, рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃџ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃА рЃАрЃљрЃбрЃћрЃАрЃбрЃЮ рЃюрЃљрЃЎрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃДрЃЮрЃЋрЃџрЃўрЃАрЃЏрЃЮрЃЏрЃфрЃЋрЃћрЃџрЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ (рЃАрЃљрЃерЃБрЃљрЃџрЃЮ, рЃЏрЃћрЃЊрЃўрЃљрЃюрЃљ, рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃБрЃџрЃћрЃЉрЃў, рЃЏрЃўрЃю/рЃЏрЃљрЃЦрЃА).

рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃЮрЃцрЃљ рЃљрЃЉрЃАрЃЮрЃџрЃБрЃбрЃБрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ (<$1K, $1K-$2.5K рЃЊрЃљ рЃљ.рЃе.) рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџрЃўрЃљ рЃўрЃЏрЃўрЃА рЃАрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ, рЃЌрЃБ рЃАрЃљрЃЊ рЃ«рЃЊрЃћрЃЉрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљ.

рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ vs. рЃљрЃарЃљрЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЊрЃдрЃћрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ: рЃерЃћрЃфрЃЊрЃЮрЃЏрЃўрЃА рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃў рЃЊрЃљрЃ»рЃњрЃБрЃцрЃћрЃЉрЃБрЃџрЃўрЃљ IsHoliday рЃЊрЃарЃЮрЃерЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌ рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃў рЃАрЃљрЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃЮ рЃЎрЃЋрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў.

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃА рЃърЃљрЃўрЃърЃџрЃљрЃўрЃюрЃў:

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃљ: рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА train, stores рЃЊрЃљ features рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃюрЃљрЃЎрЃарЃћрЃЉрЃћрЃЉрЃА Store-рЃАрЃљ рЃЊрЃљ Date-рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ.

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃњрЃљрЃгрЃЏрЃћрЃюрЃЊрЃљ: рЃљрЃњрЃЋрЃљрЃарЃћрЃЉрЃА IsHoliday рЃЊрЃБрЃЉрЃџрЃўрЃЎрЃљрЃбрЃў рЃАрЃЋрЃћрЃбрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЊрЃћрЃњрЃљрЃЊ рЃЕрЃюрЃЊрЃћрЃЉрЃљ.
